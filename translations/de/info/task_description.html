Positive ganze Zahlen können auf verschiedene Weise als Summen aufeinander folgender positiver Zahlen ausgedrückt werden. Zum Beispiel kann 42 auf vier verschiedene Arten als eine solche Summe ausgedrückt werden:(a) 3+4+5+6+7+8+9, (b) 9+10+11+12, (c) 13+14+15 und (d) 42. Wie die letzte Lösung (d) zeigt, lässt sich jede positive ganze Zahl trivialerweise immer als eine Singularsumme ausdrücken, die nur aus dieser ganzen Zahl besteht.

Berechnen Sie, auf wie viele verschiedene Arten sie als Summe aufeinanderfolgender positiver Ganzzahlen ausgedrückt werden kann.

Eingabe: Int.

Ausgabe: Int.

Beispiel:

{% if interpreter.slug == "js-node" %} 
countConsecutiveSummers(42) == 4
countConsecutiveSummers(99) == 6
{% else %} 
count_consecutive_summers(42) == 4
count_consecutive_summers(99) == 6
{% endif %}

Vorbedingung: Die Eingabe ist immer eine positive ganze Zahl.
